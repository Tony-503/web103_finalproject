# User Stories

## Outline User Roles

Customer

Admin

## Draft User Stories

1. As a customer, I want to view the arcade games available so I can decide what to play.

2. As a customer, I want to see food and drink options so I can plan my visit.

3. As a customer, I want to check upcoming events so I know when to come in.

4. As a customer, I want to save my favorite games so I can find them easily later.

5. As a customer, I want to view a leaderboard so I can compare scores with other players.

6. As a customer, I want to filter games by type so I can find games I enjoy.

7. As a customer, I want to read game descriptions so I know what each game is about.

8. As an admin, I want to add and update games so the app stays current.

9. As a customer I want to  give feedback so Arcade bar can improve the experience

10. As a customer I want to see deals so I can save money.
