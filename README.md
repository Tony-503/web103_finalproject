## About

### Description and Purpose

Arcade Bar is a fun social hangout app where guests can explore classic arcade games, food and drink options, and special events in one place. The app is designed to help users easily browse the arcade bar experience before visiting, while also giving staff a simple way to manage updates and keep the content current.

### Inspiration

The inspiration for this project came from Leaderboard Arcade, which was mentioned to me by a coworker. After visiting, I noticed that it felt different from places like Dave and Buster’s, and that experience gave me the idea to create an app for an arcade bar that feels more local, fun, and unique.
[LINK](https://leaderboardarcade.com/)
